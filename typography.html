Main typeface

Hlavní písmo 

We use our own font cesko.digital for all communication.

By a suitable combination of two font styles cesko.digital (proportional and "mono") it is possible to achieve a consistent typographic output, clearly expressing the identity and value of the community.

We use the default (proportional) font in common texts and headings. Its technical character harmonic rhythm is well suited to human messages in longer texts.

The mono version works with the same width of all characters vigorously and clearly. It suggests writing from computer terminals. Suitable for logos, short designations, as a special element. It is also easier to read in very small sizes.

Pro veškerou komunikaci používáme naše vlastní písmo cesko.digital.

Vhodnou kombinací dvou stylů písma cesko.digital (proporcionální a „mono“) lze dosáhnout souladného typografického výstupu, jasně vyjadřujícího identitu  a hodnoty komunity.

Výchozí (proporcionální) písmo používáme v běžných textech a nadpisech. Jeho technicistní charakter  a harmonický rytmus se dobře hodí na lidská sdělení v delších textech.

Verze mono působí se shodnou šířkou všech znaků rázně a přehledně. Napovídá na zápis z počítačových terminálů. Hodí se pro loga, krátká označení, jako ozvláštňující prvek. Je také lépe čitelné ve velmi malých velikostech.

Typographic tips

Typografické tipy

In plain text we use the Regular section, for Bold headings.

In normal text, line spacing of 150% of the font size is suitable for our font. This will improve the readability of each line. For example, if you are writing with a size of 11 points, set the line spacing to 16 pt.

The number of characters per line for longer texts should be somewhere between 35-60, which is optimal for readability.

Use as few levels of the hierarchy as possible. For example, if you need more than 3 levels of headings, consider a different text structure. We distinguish the headings in bold and size.

V běžném textu používáme řez Regular, pro nadpisy Bold.

V normálním textu se k našemu písmu hodí řádkování 150% velikosti písma. Zlepší to čitelnost každému řádku. Pokud například píšete velikostí 11 bodů, nastavte si řádkování na 16 b.

Počet znaků na řádek u delších textů by měl být někde mezi 35–60, což je optimální pro čitelnost.

Používejte co nejméně úrovní hierarchie. Pokud například potřebujete více než 3 úrovně nadpisů, zvažte jinou strukturu textu. Nadpisy odlišujeme tučným řezem a velikostí.

Náhradní písmo

Altrernate typeface

Náhradní písmo je ryze univerzální. Toto písmo je na každém počítači a informace, kterou napíšete v tomto písmu, se s největší pravděpodobností adekvátně zobrazí i příjemci.

The alternate typeface is purely universal. This font is on every computer, and the information you type in this font is most likely to be displayed adequately to the recipient.
